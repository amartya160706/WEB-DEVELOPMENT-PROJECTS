# 🌐 Web Development Projects Collection

This repository contains a collection of web development projects built using HTML, CSS, JavaScript, Bootstrap, and basic backend concepts.
Each project focuses on solving practical problems and demonstrating core web development skills such as UI design, form handling, validation, responsiveness, and interactivity.

---

## 📂 Projects Included

* **Box Model** – Demonstrates CSS box model concepts (margin, padding, border, content)
* **Calendar** – Dynamic calendar with date highlighting and styling
* **Customer Registration Form** – Form with input validation and structured data collection
* **Dynamic Form Validation** – Real-time validation using JavaScript
* **Email Message Interface System** – Simulated email composition interface
* **India & World Map System** – Interactive map displaying state and country information
* **International Seminar Registration Form** – Multi-field form with validation and options
* **Length Converter** – Unit conversion tool using JavaScript
* **Personal Blog** – Static blog layout with structured content
* **Responsive Webpage** – Mobile-friendly responsive design using CSS
* **Bootstrap Responsive Webpage** – Responsive UI using Bootstrap framework
* **Signup Form** – User registration form with validation
* **To-Do List** – Task management app with dynamic updates
* **Time Table** – Structured timetable using HTML tables
* **Weather App** – Displays weather data using API integration
